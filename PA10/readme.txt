PA 10 Pinball
Shawn White, Jonathan Martinez, Edward Miles

to make run the following commands from inside PA10.
 
mkdir build
cd build
cp ../makefile .
make
./Tutorial

Pinball
move the paddles to hit the ball. if the ball gets past the paddles you lose a ball if you lose all three it's game over
hitting the cylindars gives one point.

controls

 Spacebar: launch the ball out of the starting area
